⚡ **Create: Pantographs & Wires** is a mod which adds catenary wires, pantographs and more for electric modern trains in Create!

This mod breaks the rule of classic Create addons with its more realistic style and focuses on the part of the Create community that uses this mod to build impressive modern trains and railway networks. With the design and implementation inspired by [Immersive Engineering](https://modrinth.com/mod/immersiveengineering), here is a overview about the features in this mod.

## 🧩 Dependencies
- This mod requires [Create](https://modrinth.com/mod/create) or [Create Fabric](https://modrinth.com/mod/create-fabric).
- Versions with `C6` are compatible with Create 6, versions without this suffix are compatible with Create 5 (unless Create 6 is the only version available for that Minecraft version)
- Version `>= 0.2.0` require [DragonLib](https://modrinth.com/mod/dragonlib) to be installed! Older versions `<= 0.1.2` do **NOT** require DragonLib! Delete it from your mods folder. [Please read this post to see which versions of my other mods are compatible with each other](https://mrjulsen.net/mods/dragonlib/docs/blog/2026/03/01/dragonlib-3-compatibility/).

## ✅ Features
*   Various masts that can be placed in any size and rotated at an angle of 22.5 degrees (approximately)
*   Cantilevers with different sizes and insulator types
*   Catenary Wire and Power lines
*   Pantograph that automatically adjusts to the height of the catenary wire
